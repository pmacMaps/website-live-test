Featured Projects

Camp Swatara Hiking Trails Map

Camp Swatara is a faith-based family and summer camp located in southeastern Pennsylvania. The focus of this project was redesigning the hiking trails map with more accurate mapping of the trails.

Interactive Data Map

This was a custom deployment of the Esri Shortlist Storytelling web map application. It was developed for the Cumberland County Economic Development Corporation. Customizations included a toggable legend, mobile loading screen, and color branding.

SARA Report Tool

This was a custom ArcGIS Tool developed to assist with reporting for hazardous chemical facilties. The user specifies various input parameters in a form. The tool then creates risk radii, and calculates the estimated population and extracts vulnerable facilities in each radius.
