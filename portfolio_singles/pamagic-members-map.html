PAMAGIC Members Map

The PaMAGIC Members Map was developed to visualize the locations of members of the Pennsylvania Mapping and Geographic Information Consortium, provide basic information about each member, and allow users to search for members.

The web map application integrates the Leaflet.js library and the Esri Calcite Maps project. Calcite Maps is web map template based on Bootstrap. Users can search for a street address or PAMAGIC member using the Esri Leaflet Geocoding plugin.
