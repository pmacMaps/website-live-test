Imagery Comparison Map

Comparing changes to the surface of the earth over time is popular within land use planning. This could be different years of orthoimagery, land use, or land cover datasets.

The Imagery Comparison Map is a web app that allows users to compare two different years of orthoimagery using a slider widget. This provides a quick way to view changes in an area.

A form control lets users select which two years to compare. Within a single session, users can compare different scenarios. Each time a user completes the form, any existing slider widget is destroyed, and a brand new widget is created.

The technology stack for this app includes the Esri JavaScript API (4.x), Esri REST services, Bootstrap, and jQuery.
