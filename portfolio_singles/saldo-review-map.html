Subdivision & Land Development Plans Review Map

This web map app was developed for the Cumberland County Planning Commission. It shows subdivision and land development plans that have been reviewed by the Planning office. Planning staff add points to the map using a secured feature service on ArcGIS Portal. A non-editable public map service pointing to the same dataset is displayed on this web map.

By default, the app displays any plans reviewed in the current year. There are multiple ways for users to change the date filter:

  • Select a start and end date
  • Select a single year
  • Select multiple years

The technology stack for this app includes Leaflet.js, various Leaflet plugins, Esri REST services, Bootstrap, jQuery, and jQuery UI.
