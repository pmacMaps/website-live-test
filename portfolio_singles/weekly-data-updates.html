Cumberland County Weekly Data Updates

In my work with Cumberland County, I developed a template Python script that is used to automate various data update workflows. The scripts are run as Windows Scheduled Tasks on a weekly or monthly basis. The scripts write the results and/or errors of the processes to a text file. Below are some of the data updates we have automated:

  • Data replication to County departments
  • Rebuilding tiles for cached map services
  • Rebuilding address locators & overwriting geocode services
