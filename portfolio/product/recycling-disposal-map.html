Cumberland County Recycling & Disposal Map

The Cumberland County Recycling & Disposal Map application was developed to provide residents of Cumberland County with information about solid waste and recycling in their municipality. The map was built using the Leaflet.js library and several plugins. The framework for the map uses the BootLeaf project, which is a full-page map app template using Bootstrap.

Information related to the map, including a legend, are accessible from modal windows. There are several ways a user can find information about their municipality/address:

  • Locate Me widget
  • Address Search widget
  • Municipality Search widget
  • Mouse-over popup for each municipality

Users can also change the basemap, turn the thematic layers on/off, and click on thematic layers to access a pop-up with information for each feature.
